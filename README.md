# 要介護認定率の予測モデル作成

要介護認定率を予測するモデルを作成する。

# 概要

都道府県別の75歳以上高齢者人口の割合、介護施設密度(10万人当たり)、一人当たり所得、人口密度を説明変数として、
都道府県別の要介護認定率の予測モデルを作成した。

# 背景

高齢化が進行し、社会保障費が増加している。
介護保険制度は必要な制度だが、財源の確保が問題となっている。
制度を維持していくための施策を考えるにあたり、将来の地域ごとの要介護認定率の予測が参考になる。

# 使用データ

使用データは、以下の公開データを使用した。
-厚生労働省「介護保険事業状況報告（令和5年度）」「社会福祉施設等調査（令和５年度）」
-総務省統計局「人口統計」「社会・人口統計体系」「都道府県別面積（平成２５年度）」

# 分析の流れ

1. 公的統計データの収集
2. 前処理とデータ結合
3. 相関分析・可視化
4. 重回帰モデル構築
5. RandomForestRegressorとの比較
6. モデル結果の考察

## 特徴量

- 75歳以上割合
- 介護施設密度
- 一人当たり所得
- 人口密度

## 目的変数

- 要介護認定率

# 使用技術
- Python
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn

# 結果
・認定率と75歳以上人口、人口密度ともも弱い正の相関がみられた。
・一方、重回帰モデルのR²は低く、都道府県単位データのみでは認定率を十分説明できなかった
・RandomForestRegressorではさらに精度が低下し、小規模データにおける過学習の可能性がある。
