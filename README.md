# 要介護認定率の予測モデル作成
要介護認定率を予測するモデルを作成する。説明変数として、75歳以上高齢者人口の割合、介護施設密度(10万人当たり)、一人当たり所得、人口密度を使う。

# 概要
都道府県別の75歳以上高齢者人口の割合、介護施設密度(10万人当たり)、一人当たり所得、人口密度を説明変数として、要介護認定率の予測モデルを作成した。

# 使用データ
使用データは、以下の公開データを使用した。
-厚生労働省「介護保険事業状況報告（令和5年度）」「社会福祉施設等調査（令和５年度）」
-総務省統計局「人口統計」「社会・人口統計体系」「都道府県別面積（平成２５年度）」

# 分析の流れ
1. 公的統計データの収集
2. 都道府県名の統一・欠損確認
3. 各データセットをmerge
4. 人口密度・介護施設密度を算出
5. 相関分析・可視化
6. 重回帰モデル構築
7. RandomForestRegressorとの比較
8. モデル結果の考察
